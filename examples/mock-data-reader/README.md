## Example for a new data reader

This is example data for writing a new data reader based on the [`MultiFormatReader`](https://github.com/FAIRmat-NFDI/pynxtools/blob/master/src/pynxtools/dataconverter/readers/multi/reader.py) from scratch. 

This data is used directly in the how-to guide to implement a reader based on the [`MultiFormatReader`]. You can find the documentation here: [How-tos > pynxools > How to use the built-in MultiFormatReader](https://fairmat-nfdi.github.io/pynxtools/how-tos/pynxtools/use-multi-format-reader.html).
